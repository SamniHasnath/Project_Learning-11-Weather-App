# 🌦️ WeatherView 

A sleek, professional weather forecasting dashboard built with **Node.js**, **Express**, and **EJS**.

## ✨ Features
* **Real-time Weather:** Get current temperature, humidity, wind, and pressure for any city.
* **5-Day Forecast:** View upcoming weather trends with a horizontal scrollable UI.
* **Auto-Location:** One-click detection to get weather for your current GPS coordinates.
* **Search History:** Remembers your last 5 searches using local storage.
* **Modern UI:** Features a "Glassmorphism" design, Poppins typography, and a clean white theme.
* **Dark Mode:** A smooth toggle to switch between light and dark aesthetics.
* **Secure:** Uses `.env` variables to protect sensitive API credentials.

## 🚀 Tech Stack
* **Backend:** Node.js, Express.js
* **Frontend:** EJS (Embedded JavaScript), CSS3 (Flexbox/Grid/Glassmorphism)
* **API:** OpenWeatherMap API
* **Tools:** Axios (for API requests), Dotenv (for security)
